Bearder

Lightweight Tinder Frontend

What is this?

Bearder is a simple to use and lightweight remplacement for Tinder dating app. It is open source and highly customizable.

This project is still in developpement and not released yet. We will add screenshots in the bottom section bellow the more we work on it.
